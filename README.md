# AKASH Instagram Toolkit

**Creator:** Akash  
**Platform:** Termux / Linux  
**Language:** Python  

---

## 🚀 Overview

**AKASH Instagram Toolkit** is a **powerful Termux-friendly tool** with **20+ Instagram utilities** in one place.  
It allows users to:

- Check Instagram profile info  
- Generate usernames & passwords  
- Generate captions & hashtags  
- Download reels, stories, and profile pictures  
- Run IP lookups & URL shorteners  
- Check username availability  
- And much more!  

This toolkit is **easy to use, safe, and public-friendly**.
